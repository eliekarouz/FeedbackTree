FullScreenModal

fun <ScreenT : Any> FullScreenModal(content: ScreenT): ViewModal<ScreenT>

Wrap you screen in a FullScreenModal to show it as if you have hosted it in a new activity. Use cases for a FullScreenModal is when you have a split screen or even a bottom navigation view and you want the content to cover the whole screen even though the rendering is being done in a more restricted area.
